## How to Play

1. Add 3-8 players
2. Select difficulty level
3. Each round, one player is secretly the imposter
4. The imposter must answer questions while pretending to be another player
5. Everyone votes on who they think is the imposter
6. Play 5 rounds, highest score wins!

## Setup
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
